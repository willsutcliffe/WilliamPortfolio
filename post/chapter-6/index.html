Binfit - a framework for unbinned likelihood fits

January 30, 2023
  • Developed a object-orientated framework in python for performing maximum likelihood fits with sytematics handled via nuisance parameters.
  • Optimized the code for speed ensuring that the computation of the maximum likelihood is vectorized.
  • Optimization possible with several backends including iminuit and scipy optimizers.
  • Framework was utilised by around 5 publications of the Belle II and Belle experiments.
