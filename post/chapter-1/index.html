London property price prediction with data-scraping

February 1, 2023

Project summary:

  • Predicted London property prices using only scraped data with a xgb boosted regressor.
  • Scraped data for around 22,000 properties from the website right move using beautiful soup.
  • Cleaned the Data extracting many features from text using regular expressions.
  • Determined the property GPS coordinates using google maps api.
  • Feature engineered the distance to Picadilly Circus.
  • Extracted text containing total property area using property plan images.
  • Trained several regression models including a xgboost regressor and neural networks.
  • Performed a K fold grid search to optimize the hyperparameters of the xgboost regressor
  • Final best xgboost regressor has around 1% uncertainty on validation and test sets.
